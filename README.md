# Minesweeper
A basic 10x10 minesweeper game made using HTML,CSS,JS while learning from [Ania Kubow's](https://github.com/kubowania) tutotial available on [youtube](https://youtu.be/rxdGAKRndz8).

## Commands
<strong>left-click:</strong> some of the squares will disappear, some will remain blank, and some will have numbers on them if they have bomb in any of the adjacent square
<br>
<strong>right-click:</strong> adds flag to the square

#### Game gets over once you flag all bombs, which results in a win or click on any bomb, which results in loss.
